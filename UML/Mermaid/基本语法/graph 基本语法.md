#graph 基本语法



### 连线

| 符号  | 示例            | 名称         | 说明 |
| ----- | --------------- | ------------ | ---- |
| \-\-> | \-\- 描述 \-\-> | 实线指向     |      |
| ==>   | == 描述 ==>     | 加粗实线指向 |      |
| -.->  | -. 描述 .->     | 虚线指向     |      |



### 形状

| 符号 | 示例           | 名称       | 说明 |
| ---- | -------------- | ---------- | ---- |
| 缺省 |                | 默认，方块 |      |
| []   | [展示的内容]   | 方块       |      |
| ()   | (展示的内容)   | 圆弧框     |      |
| {}   | {展示的内容}   | 菱形       |      |
| (()) | ((展示的内容)) | 圆形       |      |
| \>]  | \>展示的内容]  | Odd 样式   |      |



### 特殊字符

| 符号     | 示例                           | 名称         | 说明         |
| -------- | ------------------------------ | ------------ | ------------ |
| graph    | graph TB<br/>或者<br/>graph LR | 流程定义标识 | **TB**       |
| subgraph | subgraph 子流程描述<br/>end    | 子流程       |              |
| %%       |                                |              | 不能输入中文 |
